## Layer responsibilities

| Layer | Role |
|-------|------|
| **King** | Sovereign authority; final approval for irreversible executive and commercial decisions |
| **Executive Intelligence** | Constitutional intelligence layer — governs reasoning, priorities, risk, opportunity, autonomy, and executive behaviour (EI0–EI10) |
| **Pillow** | Executive Personality — applies EI; researches; recommends; never self-amends doctrine |
| **Brain** | Orchestration and computation — dispatch, module execution, LLM routing |
| **Decision Engine** | Operational authority evaluation (L0–L4); founder-approval gating for L3/L4 |
| **Agents** | Domain-specialised workers within Brain and EI constraints |
| **Connectors** | External system integration boundary |
| **Internet** | External data and service sources |

---

## Constitutional separation

| Component | Function |
|-----------|----------|
| **Brain** | Performs computation |
| **Executive Intelligence** | Governs reasoning |
| **Pillow** | Applies Executive Intelligence |
| **Decision Engine** | Evaluates operational authority (L0–L4) |
| **Cockpit** | Presents Executive Intelligence |

Executive Intelligence is the **governing doctrine above operational implementation**. Brain, agents, and connectors implement; Executive Intelligence defines **how EmpireAI thinks**, **what it prioritizes**, and **how it behaves** at the constitutional level.
